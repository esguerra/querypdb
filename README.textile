querypdb.py is a pymol script which automatically downloads
rna structures given specific pdb query parameters and then
it performs analysis with either the information in the
structures themselves, or derived from the 3DNA software.

h1. Clone the repository

@git clone https://esguerra@github.com/esguerra/querypdb.git

h2. Usage

To run the program just call it from the prompt:

@bash-4.0$ ./querypdb.py

And it will download all structures and organize them in their
correspoding folders, and also run them through 3DNA.
